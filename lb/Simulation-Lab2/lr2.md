## Внесені зміни у вихідну логіку моделі, за варіантом

Згідно варіанту завдання, у модель **Autumn** було додано добові цикли зміни
інтенсивності сонячного світла, які також впливають на температуру середовища.
Тривалість модельної «доби» задається внутрішнім параметром.

Для цього було додано внутрішні змінні:

```netlogo
day-length
base-temperature
base-sun-intensity
temp-lag
Параметр day-length визначає кількість тактів модельного часу, що відповідає
одній добі. Значення використовується для обчислення фази доби на основі
поточного значення ticks.

Реалізовано окрему процедуру оновлення добового циклу:

to update-daily-cycle
  let phase ((ticks mod day-length) / day-length)
  let raw (sin (phase * 360 - 90))
  let daylight max list 0 ((raw + 1) / 2)

  set sun-intensity base-sun-intensity * daylight
  set sun-intensity clip sun-intensity

  let target-temp (base-temperature + (sun-intensity / 8))
  set temperature temperature + temp-lag * (target-temp - temperature)
    ...
end
Таким чином, інтенсивність сонячного світла змінюється плавно протягом доби
(день/ніч), а температура слідує за змінами сонячного випромінювання з певним
запізненням (temp-lag), що відповідає реальним фізичним процесам.

Процедура update-daily-cycle викликається на кожному такті модельного часу
у процедурі go перед оновленням візуалізації сонця.

Внесені зміни у вихідну логіку моделі, на власний розсуд
На власний розсуд у модель було додано механізм холодового стресу листя.

Для цього введено нові внутрішні параметри:

cold-threshold
cold-stress-duration
cold-detach-penalty
cold-photosynthesis-penalty
cold-stress-count
Також кожному листку додано змінну стану:

cold-stress-ticks
Якщо температура середовища опускається нижче порогового значення
cold-threshold, лист входить у стан холодового стресу на фіксовану кількість
тактів cold-stress-duration.

Під час холодового стресу:

зменшується значення attachedness, що підвищує ймовірність опадання листя;

частина sugar-level та water-level конвертується у anthocyanin, що
призводить до почервоніння листя;

у процедурі фотосинтезу (adjust-sugar) пригнічується утворення цукру за
допомогою коефіцієнта cold-photosynthesis-penalty.

Окремо реалізовано лічильник cold-stress-count, який відображає кількість
листків, що перебувають у стані холодового стресу на поточному такті
модельного часу.

Додатково змінено логіку поглинання води: при температурі нижче порогу
холодового стресу листя майже не поглинає воду, що відповідає фізіологічному
пригніченню процесів у холодних умовах.

Запропоновані зміни не порушують загальну логіку моделі, а розширюють її,
додаючи реалістичну реакцію листя на низькі температури.

Обчислювальні експерименти
Проведено обчислювальний експеримент з метою дослідження впливу тривалості
добового циклу на перебіг холодового стресу листя.

У ході експерименту змінювався параметр day-length зі значеннями:

100 тактів;

200 тактів;

400 тактів.

Інші параметри моделі залишалися незмінними. Для кожного варіанту модель
запускалася на однакову кількість тактів модельного часу.

У ході експерименту відстежувалися:

значення температури;

інтенсивність сонячного світла;

кількість листків у стані холодового стресу (cold-stress-count);

динаміка опадання листя.

Результати показали, що при меншій тривалості доби частіше виникають холодні
фази, що призводить до збільшення кількості листків у стані холодового стресу
та інтенсивнішого листопаду. При збільшенні тривалості доби зміни температури
відбуваються більш плавно, а холодовий стрес проявляється рідше.
